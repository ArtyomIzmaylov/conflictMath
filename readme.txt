-Если python не установлен, установите его https://www.python.org/downloads/

Проделать нужно следующие действия:

1)Распокуйте архив в удобную для вас папку
2)Откройте консоль cmd(командную строку)
3)Пропишите команду "cd {путь до папки куда вы распаковали"}
к примеру мы распоковали архив в по следующему пути:
C:\Саморазвитие\Программирование\Python course\conflictMath

значит пишим:
cd C:\Саморазвитие\Программирование\Python course\conflictMath

4)Пропишите следующую команду:
conflict\Scripts\activate
5)пропишите команды:
pip install pandas
pip install openpyxl
pip install numpy
pip install matplotlib

Виртуальное окружение настроено, консоль не закрывайте, перейдем к настройке входных параметров.

1)Открываем папку enterData, файл table.xlsx
Задаем в него таблицу(в точности такую, как в отчёте) в левый верхний угол на рисунке показано, как должно это выглядеть, 
ни в коем случае не добавляем лишние ячейки в этом файле. Если в итоге программа не будет работать, создайте занов table.xlsx, и поместите туда таблицу

2) Открываем текстовый файл situation и вводим только один символ(к примеру 'A'), этот символ должен соответствовать спорному вопросу.

Теперь возвращаемся в консоль

1)Пишим команду main.py
2)Переходим в папку templates, там были созданы нужные для нас таблицы и рисунки графов для определенной "ситуации".
